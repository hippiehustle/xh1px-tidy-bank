## Quick Start (5 Minutes)

### Step 1: Configure
```
1. Double-click config_gui.ahk
2. Click each bank tab (1-8)
3. Select item groups you want
4. Click "Save Bank Config"
```

### Step 2: Run
```
1. Double-click main.ahk (generated in Step 1)
2. Press F1 to start sorting
3. Press F2 for emergency stop
4. Press ESC to exit
```

That's it! Your bot is running.
