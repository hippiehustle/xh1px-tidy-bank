Issue: Executable code at global scope (for loop outside function)
Location: config_gui.ahk, lines 286-293
Original Code:
  for i := 1 to 8 {
      tabKey := "tab_" (i-1)
      if userCfg["BankCategories"].Has(tabKey) {
          tabConfigs[tabKey] := userCfg["BankCategories"][tabKey]
      } else {
          tabConfigs[tabKey] := []
      }
  }

Root Cause: AutoHotkey v2.0 does not allow executable statements at global scope
            outside of function definitions. Global scope can only contain:
            - Variable declarations
            - Function definitions
            - Class definitions
            - Comments
            - Directives
